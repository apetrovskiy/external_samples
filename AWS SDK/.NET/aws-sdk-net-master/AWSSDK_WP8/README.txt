This project uses links to reference source code in the AWSSDK_DotNet35, AWSSDK_DotNet45 and AWSSDK_WinRT projects.
Only the source code that must be different for this platform (including client interfaces and client classes) is visible in this project.

IMPORTANT: in order to build this project, you must add a reference to the NuGet package Microsoft.Net.Http (minimum version 2.1.10).
